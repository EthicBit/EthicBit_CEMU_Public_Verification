# v4.0 HSM Signing Evidence Plan

**Status:** Draft  
**Purpose:** Define the evidence required before HSM-backed or managed-key custody can be claimed.

## Evidence Targets

- HSM / CloudHSM / managed KMS provider
- Non-exportable key posture
- Signing operation evidence
- Verification evidence
- Key policy
- Rotation policy
- Access policy
- Audit logs

## Non-Claim

This plan does not claim HSM-backed custody or production-grade key custody.
