# v4.0 Managed Cloud Deployment Evidence Plan

**Status:** Draft  
**Purpose:** Define evidence required for a managed cloud deployment target.

## Evidence Targets

- Cloud provider and region
- Exact commit/tag deployed
- Container image digest
- Managed PostgreSQL evidence
- Managed monitoring evidence
- OIDC configuration
- Secrets excluded
- Rollback path
- Audit trail active

## Non-Claim

This plan does not claim managed cloud deployment validation has been completed.
