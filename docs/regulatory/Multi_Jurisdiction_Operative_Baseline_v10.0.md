# Multi-Jurisdiction Operative Baseline v10.0

Status: initial operative baseline

This document records the first more-operable multi-jurisdictional layer
for ETHICBIT / CEERV / CEMU.

Present in tree:
- `regulatory/jurisdictions/jurisdiction_registry.json`
- `regulatory/mappings/ceerv_regulatory_mapping.json`
- `regulatory/mappings/jurisdiction_coverage_matrix.json`
- `regulatory/policies/regulatory_policy_baseline.json`
- `regulatory/policies/jurisdiction_escalation_policy.json`

Current interpretation:
This layer introduces:
- an explicit jurisdiction coverage matrix
- an explicit escalation policy
- baseline recognition roles for CEERV artifacts across declared jurisdictions

It does not yet constitute:
- full multi-jurisdictional support completion,
- harmonized regulatory equivalence,
- or global regulatory certification.
