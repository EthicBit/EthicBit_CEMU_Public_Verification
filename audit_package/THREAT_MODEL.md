# Threat Model

## Primary risks
- Divergence between sovereign truth and derived reports
- Misreading historical artifacts as active truth
- External live projection failure
- Assurance-policy drift
- Signature / attestation inconsistency
- Publication or anchor desynchronization

## Trust-sensitive zones
- Official status calculator
- Mixed audience audit generation
- Live verification layer
- Assurance policy artifacts
- Final status snapshot export

## Main mitigation strategy
- Single sovereign truth artifact
- Canonical reconciliation entrypoint
- Explicit truth hierarchy
- Derived report regeneration
- Short-form active snapshot
