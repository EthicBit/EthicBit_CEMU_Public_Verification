## How to Submit a Report

1. Follow the reproduction guide: `docs/reproduction/REPRODUCTION_GUIDE.md`
2. Use the report template: `docs/reproduction/REPRODUCTION_REPORT_TEMPLATE.md`
3. For the quickstart version: `demos/aem-evolve-multi-agent-api/QUICKSTART_30_MIN.md`
4. Fork this repository.
5. Copy the filled template to this directory as `REPORT_<your-handle>_<YYYY-MM-DD>.md`.
6. Open a pull request with title: `reproduction: <your-handle> v0.3.1-demo <YYYY-MM-DD>`

Partial results and failures are accepted and valuable. Do not omit failures.

---

## Acceptance Criteria for Reports

A report is accepted if it:
- Uses the pinned commit `c634f906`.
- Was executed in a separate environment not controlled by EthicBit.
- Fills in all required fields of the template.
- Includes full output for each check (pass or fail).
- Includes an honest reproduction statement.

Reports from EthicBit employees or collaborators will be labeled `[INTERNAL]` and will not count toward the independent reproduction threshold.
